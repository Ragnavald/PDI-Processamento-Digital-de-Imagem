# PDI-Processamento-Digital-de-Imagem

Repositório dedicado aos estudos, trabalhos e implementações práticas da disciplina de Processamento Digital de Imagens (PDI). O objetivo principal é explorar técnicas de aquisição, manipulação, melhoria e análise de imagens digitais através de algoritmos computacionais, compreendendo a matemática por trás dos pixels e a sua aplicação prática.
